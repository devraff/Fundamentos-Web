# 📚 Projetos – Fundamentos da Web (UFN • 2026)

Repositório destinado ao armazenamento dos projetos e exercícios desenvolvidos durante a disciplina **Fundamentos da Web**, do curso de **Sistemas de Informação da Universidade Franciscana (UFN)** no ano de **2026**.

## 🎯 Objetivo

Este repositório tem como finalidade registrar a evolução prática no desenvolvimento web, reunindo atividades propostas em aula e pequenos projetos aplicando conceitos fundamentais de front-end.

## 🧩 Conteúdos Trabalhados

Durante a disciplina foram abordados conceitos essenciais para o desenvolvimento web, incluindo:

* Estruturação de páginas com **HTML5**
* Estilização com **CSS3**
* Layouts responsivos
* Manipulação básica com **JavaScript**
* Validação de formulários
* Boas práticas de desenvolvimento web

## 🛠 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Git
* GitHub
